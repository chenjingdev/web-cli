# Privacy Policy — agrune

**Last updated:** March 25, 2026

## Overview

agrune is a browser extension that enables AI agents to interact with web pages through annotated DOM snapshots and browser actions. This policy explains what data the extension accesses and how it is handled.

## Data Collection

agrune collects the following data **only when an AI agent actively requests it**:

- **Website content**: DOM structure and text content of the current page, converted to annotated snapshots
- **User activity**: Browser actions (clicks, scrolls, text input) are performed on behalf of the AI agent, not recorded or stored

## Data Usage

All data is:
- Processed **locally** on your device
- Transmitted **only** between the extension and the local MCP server process running on your machine
- **Never** sent to external servers, third parties, or cloud services

## Data Storage

- The extension stores only connection state and user preferences using Chrome's local storage API
- No website content or user activity data is persisted

## Data Sharing

agrune does **not**:
- Sell or transfer user data to third parties
- Use data for advertising, analytics, or credit assessment
- Collect personally identifiable information

## Permissions

The extension requests browser permissions solely to fulfill its core function of enabling AI-driven browser automation. Each permission is used only when triggered by an explicit AI agent command.
