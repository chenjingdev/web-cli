Getting Started

Follow these steps to set up your project management workspace.

1
Navigate to the Board

Click on the Board tab to view the Kanban board. Tasks are organized into columns: Backlog, In Progress, Review, and Done.

2
Create a New Task

Click the "+ New Task" button to open the task creation wizard. Fill in the task details across three steps: basic info, assignment, and review.

3
Drag and Drop Tasks

Move tasks between columns by dragging them. Click on a task card to edit its details or change its status.

4
Manage Team Members

Go to the Members tab to view and filter team members. Use pagination to browse through the member list.

5
View Documentation

The Docs tab provides access to project documentation, API references, and this getting started guide.

Tip: Data Persistence

All your changes are automatically saved to localStorage. Your tasks and board layout will persist across page refreshes.

Keyboard Shortcuts

  • N - Create new task
  • 1-4 - Switch between tabs
  • Esc - Close dialog
